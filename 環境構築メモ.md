## 3. 設定 (settings.json)

```json
{
  "markdown.copyFiles.destination": {
    "**/*.md": "images/"
  }
}
```

**変更した設定とその理由:**

- `markdown.copyFiles.destination` — Markdown ファイルからクリップボード画像を貼り付けた際の保存先を指定する。
  - キー `"**/*.md"` : 対象ファイルのパターン（すべての .md ファイル）
  - 値 `"images/"` : すべての画像を `images/` フォルダにまとめて保存
  - 例: `画像張り込みテスト.md` に貼り付けた場合 → `images/image.png`
