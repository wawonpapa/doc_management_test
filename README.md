# Doc Management Test

VSCode を使った **Markdown ドキュメント管理手法の検証** を目的としたリポジトリです。

## 概要

| 項目 | 内容 |
|------|------|
| 目的 | VSCode + Git による Markdown ドキュメント管理の検証 |
| 対象 | Markdown ファイルを中心としたドキュメント群 |
| 管理方法 | Git によるバージョン管理 |

## ファイル構成

| ファイル | 内容 |
|---------|------|
| [環境構築メモ.md](環境構築メモ.md) | VSCode のインストール・拡張機能・設定などの構築手順メモ |
| [画像張り込みテスト.md](画像張り込みテスト.md) | Markdown への画像埋め込み手順の検証 |
| [更新履歴.md](更新履歴.md) | 日単位の変更履歴 |
| [images/](images/) | 画像ファイルの格納フォルダ |

## 注意事項

- このリポジトリはテスト用途のため、内容は随時変更されます。
- 本番環境のドキュメントは含まれていません。
