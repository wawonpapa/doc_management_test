# 画像張り込みテスト

## クリップボードから画像を貼り付ける手順

1. スクリーンショットなどをクリップボードにコピーする
2. このファイルをエディタで開いた状態で、貼り付けたい行にカーソルを置く
3. `Ctrl + Shift + V` を押す
4. 保存先フォルダ・ファイル名を確認して確定する
5. 画像ファイルが自動保存され、以下のような参照が挿入される

```md
![image](images/image.png)
```
